# MTG Deck Architect

AI-powered Magic: The Gathering deck builder using Claude API.

## Features
- Natural language deck building with Claude AI
- Scryfall card search integration  
- Mana curve visualization
- Format legality checking
- Card recommendations based on strategy
- Export to MTGA, MTGO, and text formats

## Stack
- Next.js 15 (App Router)
- Claude AI (claude-opus-4-6)
- Scryfall API
- Tailwind CSS
- mana-font for MTG symbols

## Setup
1. Clone the repo
2. `npm install`
3. Set `ANTHROPIC_API_KEY` in `.env.local`
4. `npm run dev`
